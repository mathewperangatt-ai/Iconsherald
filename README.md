# IconsHerald

**Herald Yourself.**

IconsHerald is a curated editorial platform that documents distinguished
professional careers through carefully written third-person biographies.

Unlike social networks or résumé websites, IconsHerald is designed as a
permanent professional legacy archive where every published profile
undergoes identity verification, editorial review, and member approval
before publication.

## Status

🚧 This project is currently under active development.

## Technology Stack

-   Next.js 15
-   TypeScript
-   PostgreSQL
-   Prisma ORM
-   NextAuth.js (Auth.js v5)
-   Cloudflare
-   Vercel
-   Razorpay
-   Anthropic Claude API
